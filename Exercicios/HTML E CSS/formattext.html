Principais Modos de Formatação

Negrito / Destaque

Usando o Formato de negrito na frase:'Meu coelho foi atropelado'(Forma Semantica)
Agora usando a Tag <B >: 'Meu ovo esta doendo muito'(Não Semantica)

Itálico / Destaque

Agora em Italico (Forma não Semantica)"Meu sobrenome é Pintailson"
Agora da forma Semantinca "Eu moro deibaixo do coveiro de Xique-Xique Bahia"
Agora com a junção das dua Formatações "Quem tem medo não anda de Lamborghine"

Texto Marcado

Sendo possivel criar textos marcados com a tag <MARK>, como exemplo: "To com o reguinho dolorido"

Texto Grande e Pequeno

Usando a tag <small> Sendo assim, "Eu ja vo o sapo cururu"
A tag <Big> Está ABSOLETA!

Texto Deletado

E possivel marcar um texto riscado usando a tag Texto riscado

Inserido / Sublinhado

Usado para Sublinha uma palavra dando mais enfâse palavra em enfâse

Texto sobrescrito

Usado para fazer o sinal de Elevado, x30+9x-10

Texto Subscrito

Comum pra usar em formulas como H2O
